## ⚡ Stack

- **Vite 8** + TypeScript (vanilla, zero dependencies)
- **Design System**: Dark premium — Ávila Elgueta
- **Routing**: Hash-based SPA router
- **Code-splitting**: Lazy-loaded course content
- **Persistencia**: localStorage (progreso del usuario)

## 🛠️ Desarrollo Local

```bash
npm install
npm run dev    # http://localhost:5173
npm run build  # Build de producción
```

## 📦 Deploy

Este proyecto se despliega automáticamente en **Vercel** al hacer push a `main`.

```bash
# Build command: npx vite build
# Output directory: dist
# Framework preset: Vite
```
